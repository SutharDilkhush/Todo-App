# 📝 Todo List App

A simple Todo List application built to help users manage their daily tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Simple and easy-to-use interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open the `index.html` file in your web browser.

## Project Structure

```text
todo-list/
│── index.html
│── style.css
│── script.js
└── README.md
```
